# Curso

Starter workspace for the `Curso` project.

## Structure

- `src/` — TypeScript source files
- `package.json` — npm scripts and project metadata
- `tsconfig.json` — TypeScript compiler settings
- `.gitignore` — files to exclude from version control

## Quick start

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npm run build
   ```
3. Run the compiled output:
   ```sh
   npm start
   ```

## Notes

This project is configured as a simple TypeScript starter workspace. Update `src/index.ts` and the scripts to match your application requirements.
